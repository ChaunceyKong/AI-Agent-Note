# 权限系统

权限系统不是为了让 agent 更笨，而是为了让 agent 的行动先经过一道可靠的安全判断。

## 名词解释

### 查询控制平面：

`query`：系统为了完成用户当前这一次请求，而运行的一整段主循环过程，包括用户说的一句话、系统经过多轮模型调用和工具调用。

`控制平面`：负责协调、调度、决定流程怎么往下走的一层。决定“要不要继续下一轮、为什么继续、现在属于哪种继续”，这就是控制平面。

`transition`：模型转移原因，包括上一轮为什么没有结束、什么原因进行下一轮等。

### 工具控制平面

`工具控制平面`：不直接做业务结果，而是负责协调工具如何执行的一层。

`执行上下文`：工具运行时能看到的环境。

**最小工具模型**：
```text
1. ToolSpec
   模型看见的工具名字、描述、输入 schema

2. Tool Router
   根据工具名把请求送去正确的能力来源

3. ToolUseContext
   工具运行时能访问的共享环境

4. Tool Result Envelope
   把输出包装回主循环
```
